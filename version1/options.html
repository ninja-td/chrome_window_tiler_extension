Configure and quickly resize and position chrome windows to arrange your workspace. Four different window size/posision configs can be set and accessed via the shortcut (default: Ctrl + q -> 1/2/3/4).

Set the configs in the options page. The configs are specified as percentage of the total screen size.For example, to position a window spanning 70% of the screen on the right, specify the config: "30,0,70,100".

You can also resize the current window and copy the config printed in the options page.
